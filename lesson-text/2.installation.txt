Так как gulp это обычный npm пакет то мы с легкостью можем установить его через
npm.

Шаги:

1. Создаем папку для нашего gulp проекта
2. Открываем в ее через терминал
3. Выполняем команду npm init
4. Выполняем команду npm install gulp
5. выполняем команду npm install gulp-cli

Для проверки того установился ли gulp выполняем команду

> gulp --version

/ CLI version: 2.3.0
/ Local version: 4.0.2
